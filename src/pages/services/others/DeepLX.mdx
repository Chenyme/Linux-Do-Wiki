## DeepLX API

### 功能介绍
DeeplX 是一个非官方的 DeepL API 实现，用于绕过 DeepL 的官方 API 限制或提供一些额外的功能。DeepL 是一家提供高质量机器翻译服务的公司，其官方 API 需要付费订阅。DeeplX 通过非官方的方法，提供类似的翻译功能。

### 获取 Deeplx Key

点击<Link href="https://connect.linux.do/" target="_blank" style={{ textDecoration: 'none', color: '#3A9FE8' }}>Connect.Linux.Do</Link>，即可获取你的专属 Deeplx API Key

- 若担心 Deeplx API Key 被泄露，还可以在此页面进行 API Key 的更换

### 服务如何使用

#### 1. 配合 沉浸式翻译

**（1）下载并安装沉浸式翻译插件** <Link href="https://chromewebstore.google.com/detail/%E6%B2%89%E6%B5%B8%E5%BC%8F%E7%BF%BB%E8%AF%91-%E7%BD%91%E9%A1%B5%E7%BF%BB%E8%AF%91%E6%8F%92%E4%BB%B6-pdf%E7%BF%BB%E8%AF%91-%E5%85%8D%E8%B4%B9/bpoadfkcbjbfhfodiogcnhhhpibjhbnh" target="_blank" style={{ textDecoration: 'none', color: '#3A9FE8' }}>Chrome</Link> | <Link href="https://microsoftedge.microsoft.com/addons/detail/%E6%B2%89%E6%B5%B8%E5%BC%8F%E7%BF%BB%E8%AF%91-%E7%BD%91%E9%A1%B5%E7%BF%BB%E8%AF%91%E6%8F%92%E4%BB%B6-pdf%E7%BF%BB%E8%AF%91-/amkbmndfnliijdhojkpoglbnaaahippg" target="_blank" style={{ textDecoration: 'none', color: '#3A9FE8' }}>Edge</Link>

**（2）打开插件设置，选择 开发者设置-开启Beta测试特性**

<Image src="/deeplx_use1.png" width={1000} height={800} />

**（3）前往翻译服务，打开 DeepLX (Beta) 后点击去修改**

<Image src="/deeplx_use2.png" width={1000} height={800} />

**（4） 填入 Deeplx API URL**

此处的URL为：`https://api.deeplx.org/<api-key>/translate`；api-key 为刚刚的 Deeplx API Key

<br></br>

<Image src="/deeplx_use3.png" width={1000} height={800} />

#### 2. 配合 Zotero + 翻译插件

**（1）Zotero 简介**

Zotero 是一个免费的、开源的参考文献管理工具，广泛用于学术研究和写作。目前正式版是 Zotero 6，已开启下一代 Zotero public 公测，版本为 Zotero 7 。
注意6与7版本，插件不兼容。

**（2）安装Translate for Zotero插件**

插件 GitHub 仓库地址： <Link href="https://github.com/windingwind/zotero-pdf-translate" target="_blank" style={{ textDecoration: 'none', color: '#3A9FE8' }}>zotero-pdf-translate</Link>，在仓库 Release 页面下载最新版本的插件。需注意：

- For Zotero 6, install the latest stable version
- For Zotero 7, install the latest prerelease version

**（3） 打开Zotero设置，选择翻译，并填入Deeplx API URL**

翻译服务选择 DeepLX(API)，其中 密钥 为：`https://api.deeplx.org/<api-key>/translate`，`<api-key>` 为上方教你获取的 Deeplx API Key
