### 使用说明

<Callout type="info">
    此服务使用方法与 OpenAI 官方 API 一致。由于社区内有新朋友留言想要知道如何使用此服务，因此编写了此使用说明共大家了解（声明：以下为本人的使用方法，仅供参考）
</Callout>

1. 先前往 <Link href="https://api.oaipro.com/" target="_blank" style={{ textDecoration: 'none', color: '#3A9FE8' }}>api.oaipro.com</Link>，使用 Linux Do Connect 登录。

2. 由于始皇是公益给大家的，没有试用，如有需要可以去 `钱包` 内充值余额。

3. 充值完毕后前往 `令牌` 页面，点击 **`添加令牌`**，即可获取到你的 API Key。

- 接着可以前往 `聊天` 页面 - 点击左上角头像 - `应用设置` - `语言模型`，接着填入你刚刚获取的 API Key，API 代理地址为 `https://api.oaipro.com/v1`，即可开始使用。

- 也可以通过 Curl、Python、Java 进行 API 调用。具体调用方法和 OpenAI 官方一致，只需要将参数的 base_url 替换为 `https://api.oaipro.com/v1` ，  api_key 替换为你刚刚获取的 API Key 即可开始使用。
