## Refresh Token

### 功能介绍
Refresh Token 可以用来生成新的 Access Token，是OpenAI为了IOS用户登录的产物，他的**有效期都很长**，只要用户不主动修改密码，就会一直有效！

当你有了 Refresh Token ，就可以通过编写相关函数通过 Refresh Token 来获取你账号的 Access Token，而不需要自己手动获取，从而实现自动刷新 Token 的功能。

但是需要注意的是，OpenAI已对IOS登录做了限制，当同一设备登录账号过多时会封禁设备（即此苹果设备永久禁止使用 ChatGPT APP），所以请谨慎自行获取！


### 获取方法

#### 1. 通过始皇的服务接口获取
点击<Link href="https://token.oaifree.com/auth" target="_blank" style={{ textDecoration: 'none', color: '#3A9FE8' }}>前往服务接口</Link>，接口对应信任等级的额度说明如下：
- 信用等级为：0，无法使用此服务 。
- 信用等级为：1，每天获取 5 次 Access Token，无 Refresh Token 。
- 信用等级为：2，每天获取 50 次 Access Token，无 Refresh Token 。
- 信用等级为：3，每天获取 50 次 Access Token，有 Refresh Token 。
