## 如何投递 Access Token

```shell copy filename="curl"
curl -X PUT 'https://shared.oaifree.com/share/tokens' \
-H 'Content-Type: application/json' \
-d '{
    "tokens": [
        "fk-1",
        "fk-2",
        "access token"
    ]
}'
```
